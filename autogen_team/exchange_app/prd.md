# 人民币汇率显示应用 - 产品需求文档

## 1. 背景

随着全球化贸易和跨境支付的普及，个人和企业对实时汇率信息的需求日益增长。用户需要快速了解人民币与其他主要货币的兑换比率，以便做出明智的换汇决策。当前市场上缺乏一个轻量、简洁、专注于人民币汇率的工具型应用。

## 2. 目标

**核心目标**：打造一款简洁、高效、实时的人民币汇率展示工具，帮助用户快速获取主要货币汇率信息及其短期波动情况。

**关键指标**：
- 汇率数据准确性：数据源可靠，更新及时
- 响应速度：页面加载 < 3秒
- 用户满意度：界面直观，操作简单

## 3. 功能列表

### 3.1 汇率展示模块
| 功能 | 描述 |
|------|------|
| 多币种显示 | 同时展示人民币兑美元(USD)、欧元(EUR)、日元(JPY)、英镑(GBP)、港币(HKD)、韩元(KRW)共6种主要货币 |
| 实时汇率 | 显示各货币的当前买入价和卖出价 |
| 涨跌展示 | 显示24小时涨跌幅(%)和涨跌额 |

### 3.2 趋势展示模块
| 功能 | 描述 |
|------|------|
| 24小时走势图 | 为每个币种展示最近24小时的价格走势折线图 |
| 统计信息 | 显示24小时最高价、最低价、平均价 |

### 3.3 交互功能
| 功能 | 描述 |
|------|------|
| 手动刷新 | 提供刷新按钮，用户可主动触发数据更新 |
| 自动刷新 | 页面支持自动刷新机制（可选间隔：30秒/1分钟/5分钟） |
| 币种切换 | 用户可选择关注的特定币种进行查看 |

### 3.4 状态与反馈
| 功能 | 描述 |
|------|------|
| 加载状态 | 数据请求时显示加载动画 |
| 错误提示 | 网络异常或API不可用时显示友好错误信息 |
| 最后更新时间 | 显示数据最近一次更新的时间戳 |

## 4. 非功能需求

### 4.1 技术要求
- **框架**：Streamlit（Python）
- **数据源**：使用免费的公开汇率API（如 exchangerate-api 或 frankfurter.app）
- **Python版本**：3.8+

### 4.2 依赖库
```
streamlit>=1.28.0
requests>=2.31.0
pandas>=2.0.0
plotly>=5.18.0
```

### 4.3 性能要求
- 页面首次加载时间 ≤ 5秒
- 刷新操作响应时间 ≤ 3秒
- 图表渲染时间 ≤ 2秒

### 4.4 界面设计
- **主题**：支持亮色/暗色模式自动切换
- **配色**：金融类应用常用蓝色系为主色调，绿色表示上涨，红色表示下跌
- **布局**：单页垂直布局，顶部标题区 → 汇率卡片区 → 趋势图表区 → 底部刷新控制区

### 4.5 错误处理
- API请求失败：显示"数据获取失败，请检查网络连接"并提供重试按钮
- 超时处理：请求超时时间设置为10秒
- 数据异常：显示"数据异常"提示，不展示错误数据

## 5. 验收标准

### 5.1 功能验收
| 编号 | 验收项 | 通过条件 |
|------|--------|----------|
| AC-01 | 多币种显示 | 页面加载后展示6种货币的完整汇率信息 |
| AC-02 | 涨跌标识 | 上涨显示绿色，下跌显示红色，数值格式正确 |
| AC-03 | 趋势图表 | 每个币种都有对应的24小时折线图 |
| AC-04 | 手动刷新 | 点击刷新按钮后数据更新，显示新的时间戳 |
| AC-05 | 错误处理 | 模拟网络异常时显示友好错误提示而非崩溃 |
| AC-06 | 加载状态 | 数据请求期间显示加载指示器 |

### 5.2 视觉验收
| 编号 | 验收项 | 通过条件 |
|------|--------|----------|
| VC-01 | 响应式布局 | 在不同宽度窗口下布局合理，无溢出 |
| VC-02 | 色彩一致性 | 涨跌颜色与市场惯例一致（绿涨红跌） |
| VC-03 | 图表可读性 | 图表清晰显示数据点和数值标注 |

### 5.3 验收测试场景
1. **正常流程**：启动应用 → 查看6种货币汇率 → 查看趋势图 → 点击刷新
2. **网络异常**：断开网络 → 刷新数据 → 查看错误提示 → 恢复网络 → 重试成功
3. **边界测试**：快速连续点击刷新按钮，验证防抖处理

## 6. 技术实现路径

### 6.1 数据获取策略
```
数据源: frankfurter.app (欧洲央行开源API)
- 实时汇率: GET /latest?from=CNY
- 历史数据: GET /{date}..?from=CNY&to=USD,EUR,JPY,GBP,HKD,KRW
```
